Twig to the rescue!

Twig is open source, run by the guy who does Symfony which is a PHP framework. It has become to PHP what jQuery is to JS: yes there are others, but it's the clear #1.
  • Craft
  • Drupal
  • ExpressionEngine

Excellent, excellent docs to get started and get others up-to-speed
Beautiful syntax, it shares syntax with Django and many others so when I'm sharing twig with clients they're like wait: I've seen this before. Not something dreamed-up in one guy's head, it's based on many years of convention and success
